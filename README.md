🚀 About the Project

Inception is a hands-on 42 school project where I automated the deployment of a Linux infrastructure using VMs, containers, and Bash scripting.

This project highlights:

System administration & networking

Automation & scripting

Containerization & orchestration

Problem-solving in a multi-VM environment

💡 What I Built

Automated Multi-VM Deployment: Spin up multiple Linux VMs with one command.

Container & Service Setup: Launch Docker containers and configure essential services.

Networking Automation: Bridge networks, SSH access, and inter-VM communication.

Reproducible Environment: Scripts are idempotent, allowing repeatable deployments.

Monitoring & Logging: Track deployment progress and errors automatically.

Languages & Tools:
